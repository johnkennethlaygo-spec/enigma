Start Here

Fast onboarding for non-technical traders

1) Connect Wallet

Open Dashboard, click Connect Phantom, then wait for login confirmation.

Note: You do not need to enter Helius/Jupiter/admin API keys as a user.

2) Build Your Watchlist

Add up to 5 mint addresses and click Save Watchlist.

Run Scan Now to validate token quality before any execution.

3) Subscribe to Premium

Go to Premium Checkout.

  1. Choose tier.
  2. Send SOL to the shown payment address.
  3. Paste transaction signature and verify.
  4. Reconnect wallet to refresh plan badge to PRO.

4) Trading Console: Manual Test First

  1. Open the Trading Console (Manual + Agent) section.
  2. In Manual Test Flow (Paper), set decision thresholds and risk size.
  3. Run Run One Manual Test before enabling any loop.
  4. Check decisions, expected equity curve, drawdown profile, and scanner Market Regime context.

5) Autopilot Engine (PRO for Live)

  1. Configure Autopilot Execution Engine (mode, size, TP/SL/trailing, hold time).
  2. Apply an exit preset (Scalp, Balanced, Swing) if needed.
  3. Run Run One Engine Tick first to confirm behavior.
  4. Start Start Engine Loop only after stable paper outcomes.

Live mode requires PRO plan and backend live execution enabled.
